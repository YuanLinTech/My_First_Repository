Responsive Four Column Layout

Resize the browser window to see the responsive effect.
On screens that are wider than 992px, the four column blocks will be displayed in the same row.
On screens that are between 600px and 992px wide, the column blocks will be displayed in two rows, with each row having two column blocks.
On screens that are 600px wide or less, the column blocks will be stacked in four different rows on top of each other instead of next to each other.
